# SPT 单页模版

## single page template

目录介绍：

```
.
├── README.md                  # README文件
├── pages                      # 存放单页模板文件
│   └── index                  # 存放当前模板文件
│       ├── index.js           # 页面js文件
│       ├── openApi.js         # 依赖的weda cloud SDK
│       └──index.wxml          # 页面的wxml文件，支持H5和小程序通讯
└── config.js                  # 动态配置 （webviewSrc为web-view组件的src地址，envId为微搭环境ID）
```
